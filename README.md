## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🗂️ **멀티 보드** | 여러 칸반 보드를 독립적으로 생성 · 보드별 폴더 경로 · 헤더 드롭다운으로 즉시 전환 |
| 📋 **칸반 보드** | 커스터마이징 가능한 다열 구성 (기본: TO-DO / IN PROGRESS / DONE) |
| 🏷️ **해시태그** | `#태그` 형식 입력 · 자동완성 드롭다운 · 클릭 한 번으로 필터링 · 태그별 카드 수 뱃지 |
| 🔍 **전역 검색** | 제목 / 내용 / 태그 실시간 필터 · 매칭 텍스트 노란색 하이라이트 |
| 🔎 **태그별 그룹 보기** | 모든 카드를 태그 기준으로 묶어서 조회 |
| 📊 **보드 통계** | 헤더에 전체 · 완료 · 기한 초과 카드 수 실시간 표시 |
| ➕ **컬럼 커스터마이징** | 보드에서 바로 추가 / 이름 변경(폴더 자동 리네임) / 드래그로 순서 변경 · Flush 가능 여부 지정 |
| 🗃️ **보관 & 아카이브** | 보관 가능 컬럼의 카드를 일괄 아카이브, 월별로 구조화 저장 |
| 📜 **아카이브 히스토리** | 보드 내 아카이브 뷰에서 검색 · 월별 / 컬럼별 / 태그별 필터 |
| ☑️ **체크리스트** | 카드 내 항목 체크리스트 · 완료율 배지 + 진행률 바 표시 |
| 🔢 **우선순위** | 낮음 🔵 / 중간 🟡 / 높음 🔴 / ASAP 🚨 · 카드에 인라인 chip으로 표시 |
| 🚦 **WIP 리밋** | 컬럼별 최대 카드 수 설정 · 초과 시 카운트 뱃지 빨간색으로 경고 |
| 🔁 **반복 작업 스케줄러** | 매일 / 매주(요일 지정) / 매월(날짜 지정) 스케줄로 Kanban 뷰 열 때 자동 카드 생성 |
| ⚡ **빠른 카드 추가** | 컬럼 하단 입력창에서 `#태그 !높음 ^3/15` 인라인 문법으로 즉시 카드 생성 |
| 🔗 **내부 링크** | 카드 내용의 `[[노트명]]` · `[[노트명\|별칭]]` 클릭 시 해당 노트 바로 열기 |
| 🔮 **위키링크 자동완성** | 내용 편집 중 `[[` 입력 시 vault 파일 검색 드롭다운 표시 |
| ⌨️ **키보드 단축키** | `N` 새 카드 · `/` 검색 · `E` 편집 · `Del` 삭제 · `Esc` 닫기 |
| 🗒️ **카드 상세 패널** | 카드 클릭 시 오른쪽 패널에 전체 내용 표시 · 체크리스트 · 링크 · 인라인 편집 |
| 📅 **마감 임박 뷰** | 마감일 기준 그룹 보기 — 기한 초과 / 오늘 / 7일 / 30일 / 이후 |
| 🕐 **상대적 날짜** | 카드 하단에 생성일 · 수정일을 "N일 전" 형식으로 표시 |
| 🔃 **카드 정렬** | 생성일 / 마감일 / 우선순위 / 제목 기준 정렬 · 오름/내림차순 전환 |
| 🖱️ **드래그 앤 드롭** | 카드 · 컬럼 모두 드래그로 자유롭게 이동 |
| 📁 **vault-native 저장** | 카드 1개 = `.md` 파일 1개, Obsidian에서 직접 열람 가능 |

## 📦 설치

### BRAT 사용 (권장)

> 업데이트가 자동으로 적용됩니다.

1. Obsidian 커뮤니티 플러그인에서 **BRAT** 설치 및 활성화
2. BRAT 설정 → **Add Beta plugin from GitHub**
3. `Lemon-Obsidian/kanban` 입력
4. Obsidian 재시작 후 `@lemon/kanban` 플러그인 활성화

### 수동 설치

1. [최신 릴리즈](https://github.com/Lemon-Obsidian/kanban/releases/latest)에서 `main.js`, `manifest.json`, `styles.css` 다운로드
2. vault 내 `.obsidian/plugins/lemon-kanban/` 폴더에 복사
3. Obsidian 재시작 후 설정 → 커뮤니티 플러그인에서 활성화

## 🗂️ 동작 방식

### 파일 저장 구조

카드는 각 보드의 폴더 아래에 **컬럼 ID = 서브폴더** 방식으로 저장됩니다.

```
Kanban/                       ← 보드 루트 폴더 (설정에서 변경 가능)
├── todo/                     ← TO-DO 컬럼
│   ├── 디자인-시안-검토.md
│   └── API-명세-작성.md
├── in-progress/              ← IN PROGRESS 컬럼
│   └── 로그인-기능-구현.md
├── done/                     ← DONE 컬럼 (flushable)
│   └── DB-스키마-설계.md
└── _archive/                 ← Flush된 카드가 월별로 저장
    └── 2026-03/
        └── DB-스키마-설계.md

WorkKanban/                   ← 별도 보드 폴더 (멀티 보드)
├── todo/
└── done/
```

### 카드 파일 형식

각 카드는 frontmatter + 마크다운 본문으로 구성된 일반 `.md` 파일입니다.
vault에서 직접 열어 편집해도 플러그인에 자동 반영됩니다.

```markdown
---
tags:
  - 업무
  - 프로젝트A
due: 2026-03-15
priority: high
created: 2026-01-01T10:00:00.000Z
---

# 카드 제목

카드 본문 내용. [[관련 노트]]를 링크로 연결할 수 있습니다.

- [ ] 체크리스트 항목 1
- [x] 체크리스트 항목 2 (완료)
```

### 뷰 구조

```
@lemon/kanban
├── Kanban Board  (리본: columns 아이콘)
│   ├── 보드 전환 드롭다운 + [+ 보드 추가]
│   ├── [📅 마감 임박]  [🗃 아카이브]  [🔁 반복 관리]
│   ├── 검색창 (매칭 텍스트 하이라이트)
│   ├── 정렬 버튼 (생성일 / 마감일 / 우선순위 / 제목)
│   ├── 태그 필터 바 (태그별 카드 수 뱃지)
│   ├── N 전체 · N 완료 · N 기한 초과  ← 통계 바
│   ├── [N] 새 카드  [/] 검색  [E] 편집  [Del] 삭제  [Esc] 닫기  ← 단축키 힌트 바
│   ├── [TO-DO]  [IN PROGRESS]  [DONE]  ...커스텀 컬럼  [+ 컬럼 추가]
│   │    ↕ 드래그 앤 드롭으로 카드 · 컬럼 이동
│   │    + 버튼 → 카드 추가 모달
│   │    🗃 보관 (N) → 아카이브로 일괄 보관
│   │    ··· → 컬럼 이름 변경 / 삭제
│   │    ── 카드 ──
│   │       제목 [우선순위 chip]
│   │       📅 YYYY-MM-DD까지
│   │       내용 미리보기 ([[링크]] 클릭 가능)
│   │       체크리스트 (☑ 2/5, [[링크]] 클릭 가능)
│   │       #태그 #태그
│   │       생성 N일 전 · 수정 N시간 전
│   │    우클릭 → 편집 / 이동 / 삭제
│   │    ── 컬럼 하단 ──
│   │       빠른 추가 입력창 (#태그 !높음 ^3/15 제목)
│   └── (카드 hover → ✎ 편집 / ✕ 삭제 버튼)
│       (카드 클릭 → 오른쪽 상세 패널 열림)
│
├── 카드 상세 패널  (카드 클릭 시 오른쪽에 표시)
│   ├── 제목 · 컬럼 / 우선순위 배지
│   ├── 마감일 · 태그
│   ├── 전체 본문 ([[링크]] 클릭 가능)
│   ├── 체크리스트 전체 (N/M 진행률 · 인터랙티브)
│   ├── 생성일 · 수정일 상대 시간
│   └── [✎ 편집]  [E] 편집  [Del] 삭제  [Esc] 닫기
│
├── 마감 임박 뷰
│   └── 기한 초과 / 오늘 / 이번 주 / 이번 달 / 이후 그룹
│
├── 아카이브
│   └── 검색 · 월별 / 컬럼별 / 태그별 필터
│
└── 태그별 보기  (리본: tag 아이콘)
    ├── 상태 필터: 전체 / 각 컬럼
    └── 태그 그룹 (클릭으로 접기/펼치기)
         └── 카드 목록 (상태 배지 · 마감일 · 우선순위)
```

## 📖 사용법

### 보드 열기

- 왼쪽 리본의 **columns 아이콘** 클릭
- 또는 커맨드 팔레트 (`Ctrl/Cmd+P`) → `Kanban 보드 열기`

### 멀티 보드

보드 헤더의 **드롭다운**에서 보드를 전환합니다. `+` 버튼으로 새 보드를 추가하고, 설정 탭에서 각 보드의 이름과 폴더를 관리합니다.

### 카드 추가

**모달로 추가** — 컬럼 헤더의 `+` 버튼 또는 `N` 단축키

| 필드 | 설명 | 필수 |
|------|------|:----:|
| 제목 | 카드 제목 | ✅ |
| 태그 | `#업무 #프로젝트` 형식 · 입력 중 기존 태그 자동완성 드롭다운 | |
| 마감일 | 날짜 선택 (지난 날짜는 빨간색 강조) | |
| 우선순위 | 🔵 낮음 / 🟡 중간 / 🔴 높음 / 🚨 ASAP | |
| 내용 | 자유 텍스트 (`[[노트명]]`으로 내부 링크 가능, `[[` 입력 시 자동완성) | |
| 체크리스트 | `+ 항목 추가` 또는 항목 입력 후 `Enter` | |

### 빠른 카드 추가

컬럼 하단의 입력창에서 한 줄로 카드를 즉시 생성할 수 있습니다. `Enter`로 추가, `Esc`로 취소.

```
#업무 #프로젝트 미팅 준비하기 !높음 ^3/15
```

| 토큰 | 의미 | 예시 |
|------|------|------|
| `#태그` | 태그 (여러 개 가능) | `#업무 #개인` |
| `!낮음` `!중간` `!높음` `!ASAP` | 우선순위 | `!높음` |
| `^YYYY-MM-DD` | 마감일 (절대) | `^2026-03-15` |
| `^MM/DD` | 마감일 (올해 기준) | `^3/15` |
| `^오늘` `^내일` | 마감일 (자연어) | `^내일` |
| `^N일후` | 마감일 (N일 뒤) | `^7일후` |
| 나머지 텍스트 | 카드 제목 | `미팅 준비하기` |

### 반복 작업 스케줄러

보드 헤더의 **🔁 반복 관리** 버튼으로 반복 작업을 등록합니다.
**Kanban 뷰를 열 때** 스케줄이 도래한 작업의 카드를 자동으로 생성합니다. (몇 주를 안 열어도 1개만 생성)

#### 반복 주기 설정

| 주기 | 옵션 | 예시 |
|------|------|------|
| 매일 | 하루 간격 | 매일 스탠드업 노트 |
| 매주 | 요일 지정 또는 7일 간격 | 매주 목요일 주간보고 |
| 매월 | 날짜 지정 또는 30일 간격 | 매월 1일 월간 회고 |

#### 반복 작업 설정 항목

| 항목 | 설명 |
|------|------|
| 작업 제목 | 생성될 카드 제목 |
| 반복 주기 | 매일 / 매주 / 매월 |
| 요일 / 날짜 지정 | 매주: 특정 요일 · 매월: 특정 날짜 |
| 생성 컬럼 | 카드가 추가될 컬럼 |
| 우선순위 | 생성될 카드의 우선순위 |
| 마감일 오프셋 | 생성일 기준 N일 후 마감 (예: 7 → 생성일 +7일) |
| 내용 | 카드 본문 내용 |
| 체크리스트 | 한 줄에 한 항목 (생성 시 전부 미체크) |

### 위키링크 자동완성

카드 내용 편집 중 `[[`를 입력하면 vault 내 노트를 검색하는 드롭다운이 표시됩니다.

- 입력 글자로 노트 파일명 실시간 필터링 (최대 8개)
- `↑` / `↓` 탐색, `Enter` 선택, `Esc` 닫기 / 클릭도 가능
- 선택 시 `[[파일명]]` 자동 삽입 · 폴더 경로 힌트 표시

### 키보드 단축키

보드 헤더 하단의 단축키 힌트 바에서 항상 확인할 수 있습니다. 입력창이 포커스되어 있지 않을 때 동작합니다.

| 키 | 동작 |
|----|------|
| `N` | 첫 번째 컬럼에 새 카드 추가 |
| `/` | 검색창 포커스 |
| `E` | 선택한 카드 편집 (상세 패널에서 카드 선택 후) |
| `Del` | 선택한 카드 삭제 (확인 다이얼로그 · `Enter` 확인 / `Esc` 취소) |
| `Esc` | 상세 패널 닫기 |

### 카드 상세 패널

카드를 클릭하면 오른쪽에 상세 패널이 열립니다.

- 제목, 컬럼 · 우선순위 배지, 마감일, 태그 한눈에 확인
- 전체 본문 표시 (`[[링크]]` 클릭으로 연결 노트 바로 열기)
- 체크리스트 전체 표시 + `N/M` 진행률 · 체크박스 인터랙티브
- 생성일 · 수정일 상대 시간 표시
- `✎ 편집` 버튼으로 전체 편집 모달 열기

### 내부 링크

카드 내용 또는 체크리스트 항목에 `[[노트명]]` 또는 `[[노트명|표시텍스트]]` 형식으로 입력하면 클릭 가능한 링크로 렌더링됩니다.

### 태그 자동완성

태그 입력란에 `#`을 입력하면 기존 카드에서 수집된 태그 목록이 드롭다운으로 표시됩니다.

- `↑` / `↓` — 항목 탐색 · `Enter` — 선택 · `Escape` — 닫기
- 여러 태그를 공백으로 구분해 연속 입력 가능

### 카드 이동

| 방법 | 설명 |
|------|------|
| 드래그 앤 드롭 | 카드를 다른 열 위로 끌어다 놓기 |
| 우클릭 메뉴 | 이동할 컬럼 선택 |

### 카드 정렬

보드 헤더의 정렬 버튼으로 정렬 기준 선택:

| 버튼 | 기준 |
|------|------|
| 생성일 | 최신 순 (기본값) |
| 마감일 | 마감일 오름차순 |
| 우선순위 | ASAP → 높음 → 중간 → 낮음 |
| 제목 | 가나다 순 |

같은 버튼을 다시 누르면 오름/내림차순이 전환됩니다 (↓/↑ 표시).

### 체크리스트

카드 추가/편집 모달에 **체크리스트** 섹션이 있습니다.

- `+ 항목 추가` 클릭 또는 항목 입력 후 `Enter` → 새 항목 추가
- `Backspace` (빈 항목) → 항목 삭제
- 체크박스로 완료 처리 → 카드 미리보기에 `☑ 2/5` 형식으로 진행률 표시
- vault `.md` 파일에 `- [ ] 항목` / `- [x] 항목` 형식으로 저장 (Obsidian 호환)

### 마감 임박 뷰

보드 헤더의 **마감 임박** 버튼을 클릭하면 마감일이 설정된 카드를 기간별로 그룹화하여 표시합니다.

| 그룹 | 기준 |
|------|------|
| 기한 초과 | 오늘 이전 (빨간색 강조) |
| 오늘 | 오늘 마감 |
| 이번 주 (7일 이내) | 내일 ~ 7일 후 |
| 이번 달 (30일 이내) | 8일 후 ~ 30일 후 |
| 이후 | 30일 초과 |

### 컬럼 관리

| 동작 | 방법 |
|------|------|
| 추가 | 보드 오른쪽 `+ 컬럼 추가` 또는 설정 탭 |
| 이름 변경 | 컬럼 헤더 hover → `···` → 이름 변경 (폴더명 자동 리네임) |
| 삭제 | `···` → 컬럼 삭제 (카드가 있으면 삭제 불가) |
| 순서 변경 | 컬럼 헤더 드래그 또는 설정 탭 ↑/↓ 버튼 |

### 보관 & 아카이브

1. 보관 가능 컬럼의 헤더에 **`🗃 보관 (N)`** 버튼이 표시됨
2. 버튼 클릭 → 확인 다이얼로그 → **보관하기** 클릭 시 `_archive/YYYY-MM/` 폴더로 이동
3. 보드 상단 **🗃 아카이브** 버튼으로 히스토리 조회 — 검색 · 월별 / 컬럼별 / 태그별 필터

### 태그별 그룹 보기

- 리본의 **tag 아이콘** 클릭 또는 커맨드 팔레트 → `태그별 보기 열기`
- 상단 필터로 **전체** 또는 특정 컬럼 선택
- 태그 그룹 헤더 클릭으로 접기 / 펼치기

## 🛠️ 개발

```bash
npm install
npm run dev      # 개발 모드 (파일 변경 감지 + 자동 빌드)
npm run build    # 프로덕션 빌드
```

빌드 후 `main.js`, `manifest.json`, `styles.css`를 vault의 `.obsidian/plugins/lemon-kanban/`에 복사하면 즉시 반영됩니다.

### 프로젝트 구조

```
src/
├── main.ts          # 플러그인 진입점 · 뷰 등록 · 설정 탭 · vault 이벤트 처리
├── types.ts         # KanbanCard · KanbanColumn · KanbanBoard · KanbanSettings · RecurringTask 타입
├── utils.ts         # slugify · parseTags · formatTags · parseChecklist · formatChecklist · relativeTime
├── FileManager.ts   # 카드 파일 CRUD (생성 · 읽기 · 이동 · 삭제 · 아카이브)
├── KanbanView.ts    # 칸반 보드 뷰 (멀티 보드 · 검색 · 통계 · 단축키 · 상세 패널 · 반복 스케줄러)
├── CardModal.ts     # 카드 추가 / 편집 모달 (태그 자동완성 · 우선순위 · 위키링크 자동완성)
└── TagGroupView.ts  # 태그별 그룹 보기 뷰
styles.css           # 플러그인 스타일
manifest.json        # 플러그인 메타 (id: lemon-kanban)
```
